Mobile Navigation Test

Purpose: Test horizontal scrolling tabs on mobile devices

Expected: Tabs should scroll left/right, showing all text

Viewport: px wide

Simulated Mobile View (375px)

Test 1: Three Tabs (CV.HTML)

✓ Should show all 3 tabs
✓ Should scroll horizontally if needed
✓ Each tab should be fully readable

Test 2: Five Tabs (Stress Test)

✓ Should require horizontal scrolling
✓ All 5 tabs should be accessible
✓ Scrollbar should be visible

Test 3: Long Text Tabs

✓ Text should not wrap
✓ Tabs should expand to fit content
✓ Horizontal scrolling enabled

How to Test

  1. Desktop: Resize browser to < 768px wide
  2. Chrome DevTools: Press F12 → Toggle device toolbar → Select iPhone
  3. Mobile: Visit http://localhost:8080/mobile-test.html on phone
  4. Verify: Tabs scroll left/right smoothly

Debugging

If tabs are cut off or not scrolling:

  • Check: overflow-x: auto is applied to .tab-nav
  • Check: min-width: 100px (mobile) or 120px (desktop) on .tab-button
  • Check: white-space: nowrap prevents text wrapping
  • Check: flex: 1 0 auto prevents flex shrinking
