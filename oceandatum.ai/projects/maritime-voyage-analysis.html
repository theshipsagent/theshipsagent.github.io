Section 09
Methodology & Architecture
Processing Pipeline CSV ingestion (36 source files) → event deduplication → zone classification (217 zones) → ship register enrichment (52,034 ships) → voyage detection → time calculation → Phase 2 segmentation → Activity-aware draft analysis → efficiency metrics → cargo matching (FGIS grain + manifest + BOL).

Data Sources

  • MRTIS Zone Reports: 36 CSV files, 318,809 raw records
  • Terminal Dictionary: 217 zones (Facility, Activity, Cargoes)
  • Ship Register: 52,034 vessels (IMO, type, DWT, TPC)
  • FGIS Grain DB: USDA inspection certificates (DuckDB)
  • Import Manifests: Panjiva vessel/BOL records
  • Cargo BOL: Combined import/export bill of lading

Key Definitions

  • Voyage: Cross-In to Cross-Out port call cycle
  • Segment: Split at first terminal (inbound + outbound)
  • Loaded/Discharged: Draft delta + Activity classification
  • Utilization: Terminal time / total port time
  • Completeness: Both cross-in and cross-out present
  • Activity: USACE zone dictionary (Load/Discharge/L-D)
