Project Overview

This platform synthesizes multiple government data sources to provide comprehensive rail freight cost modeling, commodity flow analysis, and infrastructure assessment capabilities. The system processes STB Public Use Waybill Sample data (2005–2023), integrates URCS Phase III costing methodology, and applies network-based routing analysis to develop rate benchmarks and market intelligence for bulk commodity movements.

  • Origin-destination flow analysis by commodity (STCC classification)
  • URCS variable cost modeling with R/VC ratio calculations
  • Network routing algorithms (Dijkstra shortest path on NTAD rail network)
  • Gateway congestion analysis and infrastructure bottleneck identification
  • Class I carrier merger impact assessment (captive shipper analysis)
  • Multi-modal freight integration (rail + waterborne + truck)

Technical Architecture

Data Sources

  • STB Public Use Waybill Sample (2005–2023) — 3.2M records
  • STB URCS Phase III — Variable cost coefficients
  • BTS NTAD — North American Rail Network (197K nodes)
  • EPA GHGRP — Cement plant locations
  • BEA Economic Areas — 172 US zones + 14 Canadian regions
  • USACE Waterborne Commerce — Port tonnage cross-validation

Processing Pipeline

  • Python 3.11+ (pandas, geopandas, networkx, folium)
  • NetworkX MultiDiGraph for rail network routing
  • Haversine + Dijkstra shortest path algorithms
  • URCS variable cost: GTM + Car-Mile + Switch + Interchange
  • GeoPandas spatial joins for proximity analysis
  • Interactive visualization: Folium maps, Plotly charts

Output Formats

  • HTML reports (static, self-contained, portable)
  • GeoJSON (rail yards, routes, BEA boundaries)
  • CSV exports (processed waybill data, cost estimates)
  • Interactive maps (Leaflet/Folium with layer toggles)

Analysis Highlights

Cement Trade Flow Findings

  • Canadian Re-Origination: 27M tons enters US via CPKC at Fargo, ND (Lafarge Calgary), stored at ND terminals, re-originated as "domestic" shipments — raises true Canadian share from 2.4% to 8.7%
  • Port Import Gateway: Houston (BEA 134) handles 31M tons via Port of Houston terminals (CEMEX Mexico clinker, GCC Chihuahua, Turkish/Greek vessel imports) — largest US cement import port
  • Geographic Boundary Artifacts: Fresno (BEA 160) shows 37M tons origin but has no plants — actual Kern County plants (BEA 161) load at San Joaquin Valley yards, waybill records yard BEA not plant BEA
  • Mid-Corridor Distribution: Hartford (BEA 013) shows 14M tons — receives from Albany NY plants + Quebec imports, re-originated southward via CSX/CP to Baltimore/Charleston markets

NOLA Gateway Capacity Issues

  • Congestion: 29-mile NORG corridor handles 35 freight + 5 Amtrak trains/day at 20 mph max speed, generating 29.7 hrs/day combined train-meet delay
  • Single Point of Failure: Huey P. Long Bridge is only Mississippi River rail crossing in metro — all east-west traffic queues here
  • At-Grade Crossings: 20 crossings with 10K–20K vehicles/day generate 112.4 hrs/day vehicle delay
  • Infrastructure Needs: $4.0B+ identified investments (NORG $642M, Louisiana International Terminal $1.8B, NOGC relocation $613M)

UP-NS Merger Captive Shipper Risk

  • 2-to-1 Problem Redux: NOLA gateway eastern carriers reduce from 2 (NS + CSX) to 1 (CSX only), mirroring 1996 western 2-to-1 (UP + SP → UP only)
  • Below-Gateway Captivity: NS track south of NOLA through St. Bernard/Plaquemines becomes UP monopoly — shippers go from competitive access to UP sole carrier
  • CGP Limitation: Committed Gateway Pricing addresses through-rate competition but NOT first-mile/last-mile captivity for shippers on former NS track south of NOLA
