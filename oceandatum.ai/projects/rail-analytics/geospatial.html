Geospatial Rail Infrastructure Analysis

Rail Lines, Nodes, Yards & Facility Proximity Mapping · GeoPandas | QGIS | GeoJSON | Interactive Folium Maps

Overview

The geospatial analysis component of the Rail Analytics Platform integrates multiple authoritative datasets to map the physical rail infrastructure, identify facility clusters, perform proximity analysis between rail assets and industrial customers, and produce both static and interactive cartographic outputs. The core analysis centers on the New Orleans Rail Gateway (NORG) corridor, with national coverage for cement plant proximity analysis tied to the STB Waybill flow data.

Core Datasets Integrated

  • USDOT NTAD (National Transportation Atlas Database) — national rail lines and nodes
  • Louisiana Rail Yards dataset — NOLA gateway yard polygons and metadata
  • EPA FRS (Facility Registry Service) + GHGRP — cement plant geocoded locations
  • BEA Economic Areas — boundary shapefiles for waybill origin-destination mapping
  • USACE Waterborne Commerce — port facility locations for import terminal analysis
  • State Rail Plans (Louisiana, Texas, Mississippi) — supplemental infrastructure data

EPA Cement Plants (GHGRP)

  • 91 active US cement facilities reporting above GHGRP threshold (2023)
  • Geocoded with lat/long from EPA FRS — accuracy within 50 meters for most facilities
  • Attributes: facility name, parent company, annual clinker production, kiln count, fuel type
  • Spatial join performed against BEA Economic Area boundaries (2023 vintage)
  • Output: EPA-BEA lookup table with plant-to-BEA assignments for waybill attribution

BEA Economic Areas

  • 177 BEA Economic Areas covering the continental US (2023 boundary revision)
  • Source: US Bureau of Economic Analysis — downloadable as county-aggregation definitions
  • Boundary shapefiles derived from Census TIGER county polygons dissolved to BEA node assignments
  • Used as the primary geographic unit for origin-destination aggregation of STB waybill data

Spatial Analysis Methods

Proximity Analysis

Proximity Queries Performed

  • EPA cement plants within 5 miles of Class I mainline rail — identifies plants with practical rail access
  • EPA cement plants within 1 mile of rail yard / siding — identifies plants with direct car placement capability
  • Import marine terminals within 10 miles of rail classification yard — identifies port-rail transload potential
  • NOLA gateway yards within 2 miles of industrial facilities — maps captive shipper geography

Spatial Joins

Join Type Left Layer Right Layer Purpose
Point-in-Polygon EPA Plant Points BEA Economic Area Polygons Assign BEA code to each EPA plant for waybill matching
Buffer Intersection Rail Line Buffers (5 mi) EPA Plant Points Identify rail-accessible cement facilities
Nearest Feature EPA Plant Points NTAD Rail Nodes Find nearest rail siding or interchange for each plant

Corridor Mapping

  • NORG corridor extracted from NTAD as a 29-mile polyline subset between Bridge City and New Orleans East
  • Carrier track ownership overlaid using NTAD "RROWNER" attribute field
  • At-grade crossing locations georeferenced from USDOT FRA Highway-Rail Grade Crossing Inventory
  • Milepost annotations added from NTAD node data for infrastructure project location referencing
  • NOLA gateway flowchart produced as a schematic overlay on basemap — available upon request

Technical Tools & Methods

Python Geospatial Stack

  • GeoPandas — primary vector data manipulation, spatial joins, dissolves, and buffer operations
  • Shapely — geometry operations (intersections, buffers, nearest-point calculations)
  • Folium — interactive web map generation (HTML output with Leaflet.js backend)
  • Pyproj — coordinate reference system transformations (WGS84 ↔ Louisiana State Plane)
  • Rasterio — raster DEM processing for elevation context (NORG flood risk overlay)

QGIS Integration

  • QGIS 3.34 LTR used for final cartographic output and print layout production
  • Custom symbology styles exported as QGIS style files (.qml) for reproducibility
  • Atlas function used to generate individual carrier maps at consistent scale
  • Elevation profiles extracted using QGIS Profile Tool along NORG corridor

GDAL/OGR Command-Line Tools

  • ogr2ogr: format conversion between Shapefile, GeoJSON, GeoPackage, and PostGIS
  • ogrinfo: schema inspection and layer validation for all input datasets
  • gdal_translate: raster reprojection for DEM alignment
  • ogr2ogr -clipsrc: clipping national NTAD dataset to Louisiana/Gulf Coast study area bounding box

Use Cases

Infrastructure Planning

  • Map proposed infrastructure project locations (LIT, Almonaster Bridge, Oliver Yard lead extension)
  • Overlay flood zone boundaries (FEMA FIRM) against NORG corridor for resilience planning
  • Identify parcels needed for grade separation projects using cadastral data

Carrier Competition Analysis

  • Buffer analysis: which industrial facilities are within practical rail access distance of each carrier's mainline
  • Captive shipper mapping: identify facilities with only one carrier within economic switching distance
  • Post-merger competitive geography: visualize the change in service area overlap under proposed UP-NS transaction

Commodity Flow Mapping

  • Overlay STB waybill BEA O-D flows as proportional-width flow lines on national base map
  • Animate year-over-year changes in cement rail flow patterns (2005–2023)
  • Identify directional flow imbalances (e.g., westbound cement surplus on eastern carriers)

Client Deliverables

  • Custom facility proximity maps for cement terminal site selection studies
  • Carrier service territory maps for rate negotiation support
  • Gateway bottleneck visualization for infrastructure investment proposals
  • Interactive web maps (Folium) for client self-exploration of rail network data

Sample Outputs

Louisiana Rail Yards Map

A static A3-size (16.5" x 11.7") map showing all 7 NOLA gateway yards as colored polygons overlaid on ESRI World Imagery basemap, with carrier color coding, capacity labels, and a legend. The map includes the Huey P. Long Bridge highlighted as a bottleneck node and the proposed LIT site in New Orleans East. Available upon request.

NOLA Gateway Corridor Map

An interactive Folium web map of the full 29-mile NORG corridor with layer controls allowing users to toggle: individual carrier track ownership, rail yard polygons, at-grade crossing locations, proposed infrastructure project boundaries, and industrial facility points. Popups on each feature display operational attributes. Available upon request.

Cement Plant Proximity Analysis

A national-scale map showing all 91 EPA GHGRP cement facilities as points, sized by annual clinker capacity and colored by parent company. Buffer rings at 1 mile and 5 miles from each plant are overlaid against NTAD Class I mainlines to visualize rail accessibility. Plants outside the 5-mile buffer are identified as potential truck-only or short-line-dependent facilities. Available upon request.
